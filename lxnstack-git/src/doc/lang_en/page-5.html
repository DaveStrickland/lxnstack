Automatic alignment

To do an automatic alignment, you must select the desired alignment method from the combobox and then, after you modified its settings, press the "Align" button in the "Tools" section. The following dialog window will then appear:

In which you can chose to do a simple alignment, an alignment with derotation, a simple derotation or to reset any change you previously made.

Here there is a short description of the two automatic alignment methods available:

Phase Correlation Alignment

This type of alignment is fully automatic and no user's interventions are reuired. If there are some misalignments try to change the sharpening parameters. The phase correlation between the current image and the reference image is shown during the aligment process.

Alignment by align-points

Use the '+' and '-' buttons in the "Align points" section to add or remove an align-point.

Press the 'Automatic detection' button to automatic detect the best alignment points (see Preferences ).

Click on the image with the Right Mouse Button to move the currently selected align point or set its position in the 'X offset' and 'Y offset' boxes.

All the points must be correctly positioned on all images. If you don't want to do it manually, you can press the 'Set on all images' button.

WARNING: do not place aligment points near the image borders!

Manual alignment

A manual aligment method is also available in the "Manual alignment" section.

WARNING: the use of any of the automatic aligment methods will override any canges done in the manual aligment section. If you want to manually modify the alignment of an image then do it after the execution of the automatic aligment.

The reference image is select using the checkbox on the leftside of its name. The image to be aligned is selected by clicking its name with the left mouse button. The difference if the two images is shown in the "image viewer" box.

To shift the image use the 'X offset' and 'Yoffset' boxes. A darker difference means well aligned images.
