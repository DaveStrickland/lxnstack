Preferences

To modify the application settings, go to the menu 'Options' -> 'Preferences'

In the 'Settings' tab you can change the language of the application and some of its defaults behaving.

In the 'Alignment' tab you can modify the settings of the automatic alignment methods.

In the 'rectangle size' section you can control the size of the portion of the image, centred on an alignment point and marked by a yellow rectangle, that will be compared with the other images during the alignment process ( see Automatic alignment by points ). This rectangle will be compared by defaults only on a limited portion of the images. If there are misalignmets, try to check the box 'Use the whole image to track alignment points', but this opttion has also some sideway effects that can easly lead to the detection of "false" alignment points.

In the 'Auto detection' section you can modify the maximum number of alignment points to be detected and ist minimum quality (for more information see the opencv user's manual and search for "goodFeaturesToTrack").

WARNING : The 'automatic detection' function may detect cosmic rays and hot pixels as stars so make sure all alignment points are set correclty!

In the 'Capture devices' section you can select a capture device and modify its properties. For more information see the user's manual of the program 'v4l2-ctl'

WARNING: The OpenCV libraries cannot manage devices already in use by othe applications and this can lead to a crash. Please MAKE SURE NO OTHER PROGRAM IS USING A CAPTURE DEVICE.

WARNING: The OpenCV libraries may not handle some properties like FPS. Their controls are shown for a possible future implementation.
