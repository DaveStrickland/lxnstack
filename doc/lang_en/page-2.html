Image capture

Lxnstack is able to grab images for any device that support the video4linux interface. Activate the 'Capture' checkbox to enable the acquisition mode.

In no source device has been selected yet (see Preferences) then a dialog window will be shown in which you can select one among the available devices.

WARNING: the opencv libraries can not handle devices ALREADY IN USE and may cause the crash of the application. Please, CLOSE ANY OTHER PROGRAM THAT USES A CAPTURE DEVICE.

If the device is handeld correctly, then the capture controls will be enabled and a live view will be shown.

Press the 'Start' button to begin the capture process and press the 'Stop' button to stop it.

The maximum number of frame to acquire can be set by enabling the 'Max' checkbox and setting the number of images in the spinbox.

Press the 'Single shot' button to capture a single image.

IMPORTANT: After all desired image have been acquired, remember to disable the live view by unchecking the 'Capture' checkbox.

The captured images are saved in the image folder of the program that is ~/Pictures/lxnstack by default (see Preferences for more informations)
