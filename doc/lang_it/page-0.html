Introduzione

lxnstack è un programma ideato per allineare e sommare le immagini astronomiche (sia planetarie che del profondo cielo). Questo programma è opensource ed è distribuito secondo la licenza GNU GPL versione 3.

I requisiti necessari per il corretto funzionamento del programma sono (come già riportato nelle istruzioni di installazione):

  • 'Python 2.7' (per verificare la sua presenza digitare nel terminale il comando python2 --version)
  • Il pacchetto di python2 'numpy'
  • Il pacchetto di python2 'scipy'
  • Il pacchetto di python2 'PIL' o in alternativa il pacchetto di python2 'Pillow'
  • La versione 2 delle librerie 'opencv'
  • Le librerie Qt4
  • Il pacchetto di python2 'PyQt4'
  • Il pacchetto v4l-utils (è richiesto solo il programma v4l2-ctl)
  • Il pacchetto di python2 'PyFits' [OPZIONALE: richiesto solo per leggere/scrivere i file in formato FITS]
